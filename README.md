\## 🚀 Project Overview



The \*\*Weather Forecast \& Alert Application\*\* is a Python-based real-time system that fetches weather data and generates alerts.

\- 🔥 Heatwave Alerts  

\- 🌬️ Wind Alerts  

\- 🌫️ Weather Alerts
